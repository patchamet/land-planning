# 3D House Zone Plan — Three.js

## ไฟล์เป้าหมาย
`land-plot-house-zone-3d.html` — ไฟล์เดียว, ใช้ Three.js จาก CDN

## Scale
- จาก SVG: 7 units = 1m
- ใน Three.js: 1 unit = 1m (แปลงค่าจาก SVG ÷ 7)

## Steps

### Step 1: Scene + Camera + พื้นที่โซน ✅ Done
- สร้าง scene, camera (perspective), renderer
- OrbitControls (หมุน/ซูม)
- พื้นสี่เหลี่ยมคางหมู (11-12-14-13) สีเขียวอ่อน วางเป็นพื้น y=0
- Grid helper
- **เห็น**: พื้นที่ดินสีเขียวรูปคางหมู หมุนดูได้

### Step 2: บ้าน + ลานจอดรถ ✅ Done
- บ้าน: กล่อง 6×6×3m สีน้ำตาลอ่อน (#e8c888) ตำแหน่งตรงกับ SVG
- หลังคา: ปริซึมสามเหลี่ยม หรือ กล่องเอียง สูงเพิ่ม 2m
- จอดรถ: แผ่นบาง 6×6×0.1m สีเทา (#d0ccc4) ติดด้านบนบ้าน (ฝั่งถนน)
- **เห็น**: ตัวบ้าน+หลังคา กับลานจอดรถติดกัน บนพื้นที่ดิน

### Step 3: ต้นไม้ + แปลงผัก + ถังน้ำ ✅ Done
- ต้นไม้ใหญ่ 3 ต้น: ทรงกระบอก (ลำต้น) + ทรงกลม (พุ่ม) สีเขียวเข้ม สูง 5-8m
- ต้นผลไม้ 2 ต้น: เหมือนกันแต่เล็กกว่า สูง 3-4m
- แปลงผัก: กล่องบาง 5×5.4×0.3m สีเขียวอ่อน (#77bb44)
- ถังน้ำ: ทรงกระบอกเล็ก สีฟ้า (#8ec8e8) สูง 1m
- **เห็น**: สวนครบ มีต้นไม้ แปลงผัก ถังน้ำ รอบบ้าน

### Step 4: แนวไผ่ + ถนน + น้ำ ✅ Done
- แนวไผ่: ทรงกระบอกเรียว + พุ่มเล็ก 62 ต้น ตามแนว 3 ด้าน สูง 4-6m
- ถนน: แผ่นสีเทาอ่อน (#e8e4dc) วางเหนือโซน y=0.05
- ลำน้ำ: แผ่นสีฟ้า (#b8dff5) ตามแนว C
- สะพาน: กล่องบาง สีน้ำตาลอ่อน
- **เห็น**: ผังครบทุกองค์ประกอบ

### Step 5: Label + แสง + แดด
- Ambient light + DirectionalLight (จำลองแดดจากทิศตะวันออก)
- เงาจาก DirectionalLight
- ป้ายชื่อแต่ละส่วน (CSS2DRenderer หรือ sprite)
- ลูกศรเหนือ
- **เห็น**: มีแสงเงาสมจริง มีป้ายชื่อ หมุนดูได้ครบ

## พิกัดแปลงจาก SVG → 3D (÷7, origin shift)
- Origin 3D = node 11 (SVG 150,648) → (0, 0, 0)
- x_3d = (svg_x - 150) / 7
- z_3d = (svg_y - 648) / 7
- y_3d = ความสูง (0 = พื้นดิน)
